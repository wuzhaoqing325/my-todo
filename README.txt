本项目主要用到了以下几点技术：

1.将用户输入浏览器的数据，存储到本地，重启程序，数据也不会丢失。这里用了一个store插件来实现。

2.用了一个datetimepicker的插件，简单便捷

3.自己书写了一个“确定删除”的弹出框，在弹出框中定义了
var dfd = $.Deferred(); dfd.resolve(true); return dfd.promise(); 如果用系统提供的alert，在用户没有点击确定或者取消的情况下，程序会被阻断，不运行；在用户点击之后，程序继续运行。而使用上面的方法之后，不用等待用户点击，程序可以正常运行。

本项目过程中学习到的知识总结：

1.在自己动手写css之前，可以用normalize.css插件，预先定义一些样式

2.有些基于jquery插件的插件，在引用时，必须放在jquery插件的下面

3.js文件开头可以加一个“ ; ”,防止前面引用的文件结束时没有加该符号，而导致出现错误

4.定义div等，不一定必须在html文件中,js文件中也可以创建DOM对象，但是创建完成后要把它添加到body中

5.window有一个resize事件，当浏览器的窗口大小发生改变时，触发该事件

6.在模板中，一些类并没有添加到dom中，需要在渲染时定义一下

7.有些监听任务需要用到定时器，对事件进行循环监听

8.jQuery的extend方法可以将两个对象的一些属性或方法合并

9.e.preventDefault();阻止执行事件的默认操作
